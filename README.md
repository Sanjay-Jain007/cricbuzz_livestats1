# 🏏 Cricbuzz Clone – Live Cricket Score Web App

## 📌 Overview

This project is a Cricbuzz-inspired web application that provides live cricket match updates, scores, and match insights through a clean and interactive user interface. It simulates real-time cricket tracking using APIs and structured frontend design.

## 🚀 Features

* Live cricket scores
* Match statistics (runs, wickets, overs, strike rate)
* Match details (teams, venue, status)
* Real-time data updates using API
* Responsive UI for desktop and mobile
* Fast and user-friendly interface
* Player search functionality
* ICC rankings display
* Cricket news section
* Fantasy league module
* SQL-based analytics (for match insights)

## 🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript
API: Cricket Live Data API
Version Control: Git, GitHub

## ⚙️ Installation & Setup

1. Clone the repository
   git clone [https://github.com/your-username/cricbuzz-clone.git](https://github.com/your-username/cricbuzz-clone.git)

2. Navigate to the project folder
   cd cricbuzz-clone

3. Open index.html in your browser

## 🔌 API Integration

This project uses a cricket data API to fetch live scores, match schedules, and team details. You may need to generate your own API key depending on the provider.
